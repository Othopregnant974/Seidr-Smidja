## 💡 Lessons Learned

1. **BlenderMCP package form crashes Blender** — Must use single-file addon.py
2. **Eevee headless rendering fails on Pi 5** — GPU context error. Must use display :0 or Cycles CPU
3. **Blender 3.4 BSDF** — `Subsurface` not `Subsurface Weight`
4. **GitHub shadowban**: `runagridweaver` = 404 externally, support email bounced
5. **GitLab**: Cloudflare + reCAPTCHA blocks headless account confirmation
6. **GitHub API**: Need `user` scope for profile updates, `delete_repo` scope for repo cleanup
7. **TurboSquid FBX**: Filename has embedded space `"fbx clean.fbx"`
8. **Hermes memory DB**: 2,200 char limit — consolidate entries regularly
9. **Runa memory DB**: SQLite thread issue — can't write from delegate threads
10. **VRM export**: `BLENDER_VRM_AUTOMATIC_LICENSE_CONFIRMATION=true` must be set for headless
